# employee-review-system

## Description

A basic Employee review system

## Tech stack used

MongoDB is used for database, nodejs express as a server and ejs for views templates, javacript for coding
and html, css for designing.

# How to setup the project on local system

1. Clone this project
2. Start by installing npm if you don't have it already.
3. Navigate to Project Directory by : Using

```
cd employee-review-system

```

After reaching to the this preoject directory yo have to run this following command :

```
$ npm install
$ nodemon index.js
```

## Features

*Add/ remove/ update/ view employees
*Add/update/view performance reviews
\*Assign employees to participate in another employee's performance review

- List of performance review requiring feedback
- Submit feedback
- Login
- Register

## Directory Structure

- `/assets/css` - all css code
- `/config` - middleware mongoose passport-google-oauth passport-local- strategy
- `/controllers` - questions & option controllers code
- `/model` - review, review users.
- `/routes` - all routes according to admin and employee is here
- `/views` - all ejs files
- `index.js` - entry file
